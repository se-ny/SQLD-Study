# ✍️ SQLD 필기 시험 완벽 대비 모의고사

## 📌 Q1. 첫 10문재
* **문제:**
* 데이터 모델링의 세 가지 관점 중 '조직이 전개하는 업무와 데이터 간의 관계, 혹은 데이터 간의 관계를 정의하는 관점'에 해당하는 것은 무엇인가요?
  1. 추상화(Abstraction): 현실세계를 일정한 양식에 맞춰 간략하게 표현한다.
  2. 단순화(Simplification): 누구나 쉽게 이해할 수 있도록 제한된 표기법이나 언어로 표현한다.
  3. 명확화(Clarity): 대상의 애매모호함을 제거하고 정확하게 현상을 기술한다.
  4. 구체화(Concretization): 복잡한 현실세계를 가급적 많은 세부사항을 포함하여 상세하게 기술한다.

<details>
<summary>💡 정답 및 해설 보기 (클릭)</summary>

* **정답:** 4번
* **해설:** 데이터 모델링의 3대 특징은 **추상화, 단순화, 명확화**입니다. 복잡한 현실세계를 있는 그대로 구체화하는 것이 아니라, 단순하고 명확하게 추상화하여 표현하는 것이 핵심입니다.

</details>

## 📌 Q2. 데이터 독립성의 필요성
* **문제:** 데이터베이스에서 '데이터 독립성'을 확보했을 때 얻을 수 있는 장점과 가장 거리가 먼 것은 무엇인가요?
  1. 유지보수 비용 증가
  2. 데이터 중복 최소화
  3. 데이터 복잡도 감소
  4. 요구사항 변경에 따른 영향도 감소

<details>
<summary>💡 정답 및 해설 보기 (클릭)</summary>

* **정답:** 1번
* **해설:** 데이터 독립성을 확보하면 하위 구조가 바뀌어도 상위 애플리케이션에 영향을 주지 않으므로 변경에 유연해지고, 결과적으로 **유지보수 비용이 감소**하게 됩니다.

</details>
